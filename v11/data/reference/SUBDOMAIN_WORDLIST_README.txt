CYESCO Extended Subdomain Wordlist

This file expands the built-in OSINT subdomain discovery list used by the toolkit.
It keeps the original fast scan and adds a larger offline dictionary for better
surface mapping during authorized assessments.

How it is used:
- The OSINT subdomain scanner loads this file if present.
- The original small built-in list is still kept.
- The extended list improves discovery without removing any feature.
